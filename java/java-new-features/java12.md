# Java12

## 📌 Java 12 新特性总结表格

| **类别**          | **新特性**                                | **作用** |
|-----------------|--------------------------------|--------------------------------|
| **语法增强**    | `switch` 表达式（预览特性）    | `switch` 语句可以作为表达式返回值 |
| **GC 改进**    | G1 GC 改进                    | 减少 Full GC 触发，提高吞吐量 |
| **GC 变更**    | Shenandoah GC（实验性）        | 低延迟垃圾回收 |
| **JVM 改进**   | JVM 常量 API（JEP 334）        | 改进字节码生成 |
| **安全增强**   | TLS 1.3 完整实现               | 提高 HTTPS 安全性 |
| **性能优化**   | 低开销堆分析                   | 改进 JVM 性能分析 |
